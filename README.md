# 🌱 Análise de Dados - Base Agrofit

Projeto desenvolvido como case técnico e para fins de estudo, com o objetivo de transformar dados brutos do sistema Agrofit em uma solução analítica capaz de gerar insights sobre o mercado de defensivos agrícolas no Brasil.

---

## 🎯 Objetivo

Transformar uma base de dados operacional em um produto analítico, permitindo:

- Análise da estrutura do mercado
- Avaliação do risco ambiental dos produtos
- Identificação de padrões de uso e concentração

---

## 📚 Contexto do Projeto

Este projeto foi desenvolvido como parte de um processo de aprendizado em análise de dados e também como case técnico para vaga de Analista DataOps Jr, com foco em:

- Tratamento e qualidade de dados
- Estruturação de pipelines analíticos
- Construção de dashboards

---

## 📁 Dados

- Os arquivos originais são muito grandes para upload no GitHub.

- Foi disponibilizada uma amostra dos dados para fins de visualização.

O dataset completo pode ser acessado em:
https://dados.gov.br/dados/conjuntos-dados/sistema-de-agrotoxicos-fitossanitarios-agrofit

---

## ⚙️ Tecnologias Utilizadas

- Python (pandas)
- Power BI
- Excel / CSV
- PowerPoint (PPT)

---

## 🧹 Tratamento de Dados

Etapas realizadas em Python:

- Padronização de colunas (lowercase, trim)
- Remoção de duplicados
- Tratamento de valores nulos
- Normalização de textos (empresas e culturas)
- Exportação da base tratada

---

## 🔄 Pipeline de Dados

Fluxo do projeto:

Python → CSV → Power BI → PowerPoint

- Tratamento e limpeza dos dados
- Estruturação da base analítica
- Criação de métricas e visualizações

---

## 📊 Dashboard

O dashboard foi desenvolvido no Power BI para análise interativa dos dados, contendo:

- Total de produtos registrados
- Percentual de risco ambiental
- Participação das principais empresas
- Distribuição por cultura e categoria de risco

## 📈 Principais Insights

- Cerca de **72% dos produtos** apresentam alto risco ambiental
- Os **3 maiores players representam aproximadamente 18% dos registros**, indicando baixa concentração de mercado
- Forte presença de culturas como soja e milho na distribuição dos produtos

Possíveis melhorias futuras:
